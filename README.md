## Getting Started

### 0. Clone 및 패키지 설치

```
git clone https://github.com/yuda1124/feed-system
cd feed-system
yarn
```

### 1. 환경변수 설정

- .env.sample 파일 참조

```jsx
REACT_APP_END_POINT=${END_POINT}/${PREFIX}
ex) REACT_APP_END_POINT=http://a.tv/api
REACT_APP_IMAGE_PATH=
```

### 2. start

```jsx
yarn start
```
